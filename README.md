Sudo-Soldiers-Website
=====================

The website for UW's CSE side project club (students.washington.edu/uwsudo/)

How to start
------------

 1. Open terminal
 2. $ python -m SimpleHTTPServer
 3. Open 0.0.0.0:8000 in Chrome

How to contribute
-----------------

 1. Fork the repo
 2. Make changes
 3. Make a push request
